# 스마트 농업 사이트 (React + Vite)

이 저장소는 첨부된 이미지 레이아웃을 참고한 반응형 리액트 템플릿입니다.

빠른 시작 (Windows PowerShell):

```powershell
cd C:\Users\Com\Desktop\web
npm install
npm run dev
```

- 파일: `index.html`, `src/main.jsx`, `src/App.jsx`, `src/components/Hero.jsx`, `src/components/Card.jsx`, `src/styles.css`
- UI: 헤더, 히어로 섹션(배경 이미지/블러), 3개의 카드(하우스 시공, 팜, 농자재)

원하시면 이미지 교체, 다국어, 라우팅(React Router) 등 추가로 구현해드릴게요.
