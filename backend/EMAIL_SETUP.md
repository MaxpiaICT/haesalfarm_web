# 네이버 이메일 인증 설정 가이드

## 1. 네이버 앱 비밀번호 생성

1. 네이버 메일 로그인: https://mail.naver.com
2. 설정 → 메일 환경설정 → 보안 → 2단계 인증 설정
3. 앱 비밀번호 생성:
   - 네이버 계정 설정 → 보안 → 2단계 인증 → 앱 비밀번호
   - 앱 이름: "햇살농업건설 서버" (또는 원하는 이름)
   - 생성된 16자리 앱 비밀번호를 복사

## 2. 환경 변수 설정

`backend` 폴더의 `.env` 파일에 다음 변수를 추가하세요:

```env
# 네이버 이메일 설정
EMAIL_USER=your-email@naver.com
EMAIL_PASSWORD=your-app-password-16-digits
```

**중요:**
- `EMAIL_USER`: 네이버 이메일 주소 (예: haesalfarm@naver.com)
- `EMAIL_PASSWORD`: 네이버 앱 비밀번호 (16자리, 일반 비밀번호가 아님)

## 3. 이메일 인증 기능

### 회원가입 시 이메일 인증
- 회원가입 시 이메일 입력 필수
- "인증코드 발송" 버튼 클릭
- 이메일로 받은 6자리 인증 코드 입력
- "인증확인" 버튼 클릭하여 인증 완료
- 인증 완료 후 회원가입 가능

### 인증 코드 유효 시간
- 인증 코드는 10분간 유효
- 인증 코드 발송 후 3분간 재발송 제한

### 비밀번호 찾기
- 비밀번호 찾기 시 임시 비밀번호가 이메일로 발송됩니다.

## 4. 테스트

1. 백엔드 서버 재시작
2. 회원가입 페이지에서 이메일 입력
3. "인증코드 발송" 버튼 클릭
4. 이메일 확인 후 인증 코드 입력
5. "인증확인" 버튼 클릭

## 문제 해결

### 이메일 발송 실패
- `.env` 파일의 `EMAIL_USER`와 `EMAIL_PASSWORD` 확인
- 네이버 앱 비밀번호가 올바른지 확인 (일반 비밀번호가 아님)
- 네이버 2단계 인증이 활성화되어 있는지 확인

### 인증 코드가 오지 않음
- 스팸 메일함 확인
- 이메일 주소가 올바른지 확인
- 서버 로그 확인 (`console.log`로 이메일 발송 상태 확인)
