# 이메일 발송 설정 가이드 (네이버 메일)

비밀번호 찾기 기능에서 임시 비밀번호를 이메일로 발송하려면 네이버 메일 설정이 필요합니다.

## 네이버 메일 SMTP 설정 방법

### 1단계: 네이버 메일 환경 설정

1. 네이버 메일 접속: https://mail.naver.com
2. **환경설정** → **POP3/IMAP 설정** 메뉴로 이동
3. **POP3/IMAP 사용함** 체크
4. **보안 메일** 설정 (선택사항, 보안 강화)

### 2단계: 백엔드 환경 변수 설정

`backend/.env` 파일에 다음 내용을 추가하세요:

```env
# 이메일 발송 설정 (네이버 메일)
EMAIL_USER=your-email@naver.com
EMAIL_PASSWORD=your-naver-password
```

**중요:**
- `EMAIL_USER`: 네이버 메일 주소 전체 입력 (예: `your-email@naver.com`)
- `EMAIL_PASSWORD`: 네이버 계정 비밀번호 또는 메일 전용 비밀번호

### 3단계: 서버 재시작

환경 변수를 추가한 후 백엔드 서버를 재시작하세요:

```bash
cd backend
npm run dev
```

## 네이버 메일 전용 비밀번호 설정 (권장)

보안을 위해 메일 전용 비밀번호를 사용하는 것을 권장합니다:

1. 네이버 계정 설정: https://nid.naver.com/user2/help/myInfo
2. **보안 설정** → **2단계 인증** 활성화
3. **앱 비밀번호** 또는 **메일 전용 비밀번호** 생성
4. 생성된 비밀번호를 `EMAIL_PASSWORD`에 설정

## 테스트

1. 비밀번호 찾기 페이지에서 아이디와 이메일 입력
2. "임시 비밀번호 발급" 클릭
3. 입력한 네이버 메일 주소로 임시 비밀번호가 발송됩니다
4. 이메일을 확인하여 임시 비밀번호 확인

## 문제 해결

### 이메일이 발송되지 않는 경우

1. **환경 변수 확인**: `backend/.env` 파일에 `EMAIL_USER`와 `EMAIL_PASSWORD`가 올바르게 설정되었는지 확인
2. **POP3/IMAP 설정 확인**: 네이버 메일에서 POP3/IMAP 사용이 활성화되어 있는지 확인
3. **비밀번호 확인**: 네이버 계정 비밀번호가 올바른지 확인
4. **서버 로그 확인**: 백엔드 서버 콘솔에서 이메일 발송 오류 메시지 확인

### 네이버 메일 보안 설정

- 네이버는 보안상의 이유로 외부 앱 접근을 제한할 수 있습니다
- **보안 메일** 설정을 활성화하면 추가 보안 절차가 필요할 수 있습니다
- 문제가 지속되면 네이버 고객센터에 문의하세요

## 다른 이메일 서비스 사용

Gmail, Outlook 등 다른 이메일 서비스를 사용하려면 `backend/utils/email.js` 파일의 `createTransporter` 함수를 수정하세요.
