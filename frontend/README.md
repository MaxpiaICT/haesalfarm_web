# 프론트엔드 (React + Vite)

햇살농업건설 웹사이트 프론트엔드 애플리케이션입니다.

## 설치 및 실행

```bash
npm install
npm run dev
```

프론트엔드는 `http://localhost:5173`에서 실행됩니다.

## 환경 변수 설정

`.env` 파일을 생성하고 다음 내용을 추가하세요:

```env
VITE_API_URL=http://localhost:3001/api
VITE_USE_LOCAL_STORAGE=false
```

## 빌드

```bash
npm run build
```

빌드된 파일은 `dist/` 폴더에 생성됩니다.
